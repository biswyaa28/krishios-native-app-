# Security Policy

We take security seriously and appreciate efforts to report vulnerability issues responsibly.

---

## 1. Supported Versions

Security updates are actively applied to:
* Version 1.0.x (Current stable branch)

---

## 2. Reporting a Vulnerability

Please **DO NOT** report security vulnerabilities via public GitHub issues.

Instead, report security bugs privately by emailing the maintainer team at `security@krishios.org`.

Please include:
* A detailed description of the vulnerability.
* Step-by-step instructions to reproduce the issue.
* Any proof-of-concept scripts or screenshots.

We will acknowledge receipt of your report within 48 hours and work with you to release a patch.
