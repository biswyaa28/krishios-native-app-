Key Decisions

1

Riverpod over Provider/Bloc

Compile-safe, no BuildContext needed, better testability. Modern Flutter standard.

2

Hive over SQLite/SharedPrefs

Fast NoSQL, type-safe, offline-first. Perfect for scan history and user prefs.

3

Firebase + Cloud Functions

Auth, Firestore for community, Cloud Functions for weather alerts & analytics processing.

4

Clean Architecture (lite)

data/domain/presentation layers per feature. Not over-engineered, but separated.
