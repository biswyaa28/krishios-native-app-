Improvement Roadmap

Prioritized phases for demo/pitch readiness — 4 weeks

Phase 1 — Foundation (Week 1-2)

Set up architecture so everything else builds cleanly on top.

A

State Management with Riverpod

Provider-based architecture. Testable, scalable, modern Flutter standard.

B

Local Persistence (Hive)

Scan history, user preferences, offline cache. Fast NoSQL key-value store.

Phase 2 — Backend & Community (Week 2-3)

Firebase integration for real data and user accounts.

C

Firebase Auth + Firestore

Google sign-in, email auth, real-time database for community posts.

D

Live Community Feed

Real posts, comments, likes. User profiles, expert badges, categories.

Phase 3 — Weather & Analytics (Week 3-4)

Enhance existing features with real data and deeper insights.

E

Enhanced Weather Dashboard

7-day forecast, farming alerts (frost, rain), historical data, soil conditions.

F

Real Analytics Dashboard

Scan history trends, field health over time, yield predictions, exportable reports.
